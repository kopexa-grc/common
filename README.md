# Common Library

A Go library providing common utilities and functionality for Kopexa GRC applications.

## Features

- Common utilities and helper functions
- Standardized error handling
- Shared data structures
- Common interfaces

## Installation

```bash
go get github.com/kopexa-grc/common
```

## Usage

```go
import "github.com/kopexa-grc/common"

// Example usage
```

## Development

### Prerequisites

- Go 1.24 or later
- Git
- Make (optional)

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/kopexa-grc/common.git
   cd common
   ```

2. Install dependencies:
   ```bash
   go mod download
   ```

### Testing

Run the test suite:
```bash
go test ./...
```
